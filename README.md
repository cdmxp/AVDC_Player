### 参数说明
* pic_type(显示图片种类): poster or thumb <br>
    * poster: 封面图; thumb: 缩略图 <br>
* show_info(鼠标悬浮显示信息): 1 or 0 <br>
    * 1: 显示; 0: 不显示 <br>
* movie_path(视频路径): 界面左侧会显示**movie_path**下的**两层目录**。
